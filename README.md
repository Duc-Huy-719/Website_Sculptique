**Pixel-Perfect Landing Page – Bài Test Frontend**

**Mô tả**

Repository này chứa phần triển khai front-end với độ chính xác cao theo từng pixel của trang e-Commerce tham chiếu sau:

Trang tham chiếu:
https://trysculptique.com/products/lymph-cc-select

* Mục tiêu của bài test là đánh giá:

* Chất lượng triển khai giao diện front-end

* Độ chính xác UI

* Cách tổ chức và cấu trúc code
**Live Demo**
  
Website demo:
https://drive.google.com/drive/folders/1p7yQcTuqVL750wmJFhtlxjA-WtSAzmSj
**Công nghệ sử dụng**
* HTML5

* Tailwind CSS

* JavaScript thuần (ES6)

Không sử dụng backend hoặc thư viện UI bên thứ ba.

**Phạm vi triển khai**

Đã thực hiện:

*  Layout chính xác theo pixel (spacing, typography, màu sắc, tỷ lệ font)

* Giao diện responsive (desktop, tablet, mobile)

* Cấu trúc component rõ ràng

**Ghi chú kỹ thuật**

* Layout được xây dựng bằng flexbox và CSS grid tùy theo hành vi nội dung

* Breakpoint responsive được xử lý rõ ràng để tránh sai lệch giao diện

* Các giá trị UI (khoảng cách, font size, line-height) được đối chiếu trực tiếp với trang tham chiếu

* JavaScript chỉ dùng cho hành vi UI, không phụ thuộc framework
